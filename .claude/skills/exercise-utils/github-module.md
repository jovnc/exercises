# github-module.md

## Overview
Wrapper functions for GitHub CLI (gh) commands.

**When to use**: For GitHub operations requiring authentication (create/delete repos, forks, etc.).

## Functions
**Core operations**: repo management, forking, cloning, user queries

Full API: [exercise_utils/github_cli.py](../../../exercise_utils/github_cli.py)
